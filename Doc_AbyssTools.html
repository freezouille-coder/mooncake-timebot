🚀 Auto Shader Tool

L'outil principal pour importer vos assets dans Maya. Il gère la conversion, le nettoyage, le shading automatique et la configuration ACES.

Interface Principale v10

Fonctionnalités Clés

  • Import Multi-Formats : Accepte les fichiers .glb (Rodin/IA), .fbx et même les images .png (crée un plan 3D automatiquement).
  • Conversion Invisible : Utilise Blender en arrière-plan pour convertir les fichiers GLB complexes en FBX propres pour Maya.
  • Auto-Shading : Scanne le dossier source, détecte les textures (BaseColor, Normal, Roughness, etc.) et crée un réseau Arnold Standard Surface complet.
  • ACES Ready : Configure automatiquement les espaces colorimétriques (sRGB pour la couleur, Raw pour les maps techniques).

Comment l'utiliser ?

  1. References Project : Utilisez les boutons violets pour importer le LightRig ou le personnage Gloops en un clic.
  2. Import :
    • Cochez "Auto-Shading après Import".
    • Cliquez sur le bouton orange IMPORTER.
    • Sélectionnez votre fichier. Laissez le script travailler (barre de chargement).
  3. Shading Manuel : Si vous avez déjà l'objet, sélectionnez le dossier de textures dans le champ "Chemin des textures" et cliquez sur le bouton bleu en bas.

📜 Nomenclature des Textures

Pour que l'AutoShader fonctionne "magiquement", les fichiers de textures doivent impérativement suivre ces règles de nommage.

NomAsset_Suffixe.UDIM.Extension

Les Suffixes Requis

Type de MapSuffixe ExactUsage
Albedo / Couleur_BaseColorCouleur diffuse (sRGB)
Normal Map_NormalDétails de relief (Raw)
Packed Map_RGBARoughness + Metal + SSS (Raw)
Displacement_DSHauteur / Height (32bit EXR)
Emission_EmitLumière (sRGB)

La Map "Packed" (_RGBA)

Nous utilisons une texture optimisée qui combine 3 canaux en un seul fichier :

Red   ➔ Roughness
Green ➔ Metalness
Blue  ➔ SSS (Subsurface)

🎨 Smart Material Palette

L'outil ultime pour le Set Dressing et le Blocking. Il permet d'assigner rapidement des couleurs et des matériaux "Cartoon" cohérents.

Fonctionnement

  1. Choisir le Type : Sélectionnez Plastic (Standard), Metal (Brillant/Satiné) ou Soft (Matte/Gomme) en haut.
  2. Assigner : Sélectionnez vos objets (Mesh ou Curve) et cliquez sur une couleur dans la grille.
  3. Viewport vs Rendu : Le script crée un shader hybride : un Blinn optimisé pour le viewport et un Arnold Standard correct pour le rendu.

Outils Avancés

  • Random Color : Applique une couleur différente au hasard sur chaque objet sélectionné.
  • Update Scene : Met à jour tous les shaders existants sans perdre les assignations.
  • Merge Duplicates : Fusionne les shaders identiques.
  • Custom Colors : Importez/Exportez vos palettes via [IMP]/[EXP].

🍽️ Pause Déjeuner Auto

Par défaut, 1h est déduite automatiquement si ta session > 6h.

Si tu fais /pause de 30min+ toi-même → pas de déduction auto (le bot considère que tu as géré).

Configure avec /mylunch 45 (45min) ou /mylunch 0 (désactivé).

Exemple : /start à 9h, /stop à 18h, pas de /pause
→ Brut: 9h — Lunch auto: -1h → Payé: 8h
