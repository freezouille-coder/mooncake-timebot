👀 Suivi en temps réel

CommandeDescription
/who👀 Qui est en ligne maintenant (rôle + département + heures)
/today [dept]📋 Résumé du jour : travail, overtime, off, dailies
/dailies [dept] [date]📋 Dailies + manquants + liens vers posts

📊 Rapports Mensuels

CommandeDescription
/summary [mois] [année] [dept]📊 Résumé par département : heures, overtime, rôles, montants
/report [mois] [année] [dept]📑 Export TXT + CSV (colonnes : Rôle, Heures sup, Taux, Montant...)

💰 Taux Horaires

💡 Taux par défaut selon le rôle : Head=50$/h, Lead=40$/h, Artist=25$/h... Le /setrate override toujours le taux du rôle.
CommandeDescription
/setrate @artiste [taux] [devise]💰 Taux custom (override le défaut du rôle)
/rates💰 Tous les taux de l'équipe

TXT — rapport lisible, groupé par département, avec rôles et overtime

CSV — séparateur ;, colonnes :

Artiste · Rôle · Département · Date · Début · Fin · Pause · Heures · Heures sup · Taux · Montant · Devise · Daily · Message · Lien · Type

🏖️ Congés & Vacances

Demande individuelle

1. Artiste : /conge 14/04/2026 18/04/2026 "Vacances"
2. Bot poste dans #time-tracking-admin avec @Admin + emojis ✅ ❌
3. Admin, Lead, Supervisor ou Head du même dept clique ✅ ou ❌
4. Artiste notifié dans son canal -progress
5. Le matin des congés → message dans le canal progress
💡 Permissions graduelles : un Lead Animation peut approuver les congés d'un Artist Animation, même sans le rôle @Admin.

Vacances collectives

/vacances 15/03/2026 28/03/2026 "Vacances de printemps"
→ Tout @DreamTeam off. Un artiste peut /start quand même.

/cancelvacances 15/03/2026 28/03/2026

🔔 Automatisations

📅 Matin

Notif congé matin

Message dans -progress des artistes en congé.

Rappel /start +1h

Boutons 🟢 LIVE · 🏖️ OFF · ⏰ En retard. Respecte jours/horaires de l'artiste.

📝 Journée

Rappel daily +7h

Si travaillé mais pas de #daily.

Session oubliée 30min

À end_hour : rappel. >10h : alerte admin dans #time-tracking.

🌆 Soir

17h — Notif fériés

Prévient dans #time-tracking si demain est férié.

20h — Résumé auto (/today)

Résumé complet + rappel daily insistant aux retardataires. /stop bloqué sans daily.

23h55 — Résumé final

🦉 Night Owl

Minuit — "Tu bosses encore ?" Boutons: Ozef / Ok c'est fini
3h — Fermeture forcée 🛏️ Message marrant + auto-daily
00h05 — Fermeture sessions oubliées veille

⚙️ Configuration

Constantes (bot.py)

ADMIN_ROLE_NAME = "Admin"
TEAM_ROLE_NAME = "DreamTeam" # Rôle équipe
DEPT_ROLE_SUFFIX = "Team" # Ex: "Animation Team"
SUMMARY_CHANNEL_NAME = "time-tracking"
ADMIN_CHANNEL_NAME = "time-tracking-admin" # Privé admin
PROGRESS_CHANNEL_SUFFIX = "-progress"
DAILY_KEYWORD = "#daily"
DEFAULT_TIMEZONE = "CET" # Paris/Madrid
DEFAULT_SCHEDULE_START = 10 / END = 18
REMINDER_DAILY_OFFSET = 7 # start + 7h

🚀 Installation

1. Bot Discord

discord.com/developers → New Application → Bot → Token

Intents : ✅ Server Members, ✅ Message Content

OAuth2 : bot + applications.commands

Permissions : Send Messages, Embeds, Attach Files, Read History, Add Reactions

2. Railway

Push sur GitHub → railway.app → Deploy from GitHub

Variable : DISCORD_BOT_TOKEN=ton-token

3. Local

pip install -r requirements.txt
export DISCORD_BOT_TOKEN="ton-token"
python bot.py
